# Time Tracker

Simple time tracking for Obsidian.

## Features

- Start/stop timer by project
- Quick-add minutes with + button
- Weekly/monthly analytics with stacked bar chart
- Streak tracking (consecutive days above target)

## Install

1. Copy `main.js`, `styles.css`, `manifest.json` to `.obsidian/plugins/time-tracker/`
2. Enable in Settings → Community plugins

## Usage

- Click project pill to start timer
- Click + to add minutes manually
- Click status bar to open sidebar
- Configure projects and streak target in Settings → Time Tracker
